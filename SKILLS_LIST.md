# 📚 Catalogo Skill Antigravity

Questo documento raggruppa le skill importate in categorie logiche per facilitarne la consultazione e l'utilizzo.

## 🤖 AI, LLM & Agenti Autonomi
Skill dedicate allo sviluppo, orchestrazione e ottimizzazione di agenti intelligenti e integrazioni con modelli di linguaggio.

### Agenti & Orchestrazione
- `agent-evaluation`: Metodologie per valutare le performance degli agenti AI.
- `agent-manager-skill`: Gestione operativa degli agenti.
- `agent-memory-*`: Sistemi di memoria a breve e lungo termine per agenti (incluso MCP).
- `agent-orchestration-*`: Strategie per il coordinamento multi-agente.
- `agent-tool-builder`: Creazione di tool personalizzati per agenti.
- `autonomous-agents`: Pattern per agenti autonomi.
- `crewai`: Framework per l'orchestrazione di team di agenti AI.
- `langchain-*`: Architetture e pattern per LangChain.
- `multi-agent-*`: Pattern e brainstorming per sistemi multi-agente.
- `subagent-driven-development`: Metodologie di sviluppo guidate da sotto-agenti.

### LLM & Prompt Engineering
- `llm-app-patterns`: Pattern architetturali per applicazioni LLM.
- `llm-evaluation`: Tecniche di valutazione dei modelli.
- `prompt-engineering-*`: Tecniche avanzate di prompting e ottimizzazione.
- `rag-*`: Implementazione di Retrieval-Augmented Generation.
- `gemini-api-*`: Integrazione e sviluppo con Google Gemini.
- `claude-*`: Skill specifiche per l'ecosistema Anthropic Claude.
- `azure-ai-*`: Vasta suite di skill per i servizi AI di Azure (OpenAI, Vision, Speech, ecc.).

## ☁️ Cloud, DevOps & Infrastructure
Skill per la gestione dell'infrastruttura, deployment e CI/CD.

### Azure & Microsoft
- `azure-*`: Collezione completa per servizi Azure (CosmosDB, Functions, Storage, EventHub, ecc.).
- `azd-deployment`: Deployment con Azure Developer CLI.

### AWS & Altri Cloud
- `aws-*`: Cost optimization, serverless, e penetration testing su AWS.
- `terraform-*`: Infrastructure as Code con Terraform.
- `kubernetes-*`: Architettura e deployment su K8s.
- `docker-expert`: Gestione avanzata di container Docker.
- `vercel-*`: Automazione e deployment su Vercel.
- `cloudflare-workers-expert`: Sviluppo edge con Cloudflare.

### CI/CD & Automation
- `cicd-automation`: Automazione delle pipeline di integrazione continua.
- `github-actions-*`: Workflow e automazioni per GitHub.
- `gitlab-ci-patterns`: Pattern per GitLab CI.
- `jenkins`, `circleci`: (Se presenti) Automazione build.

## 🛡️ Security & Ethical Hacking
Skill offensive e difensive per garantire la sicurezza delle applicazioni.

### Offensive Security (Red Team)
- `pentest-*`: Checklist e comandi per penetration testing.
- `web-security-testing`: Test di vulnerabilità web (XSS, SQLi, ecc.).
- `api-security-testing`: Test di sicurezza per API.
- `red-team-*`: Tattiche e strumenti per simulazioni di attacco.
- `metasploit-framework`: Utilizzo del framework Metasploit.
- `burp-suite-testing`: Test con Burp Suite.
- `shodan-reconnaissance`: Ricognizione con Shodan.

### Defensive Security (Blue Team)
- `security-audit`: Audit di sicurezza generali.
- `codebase-audit-pre-push`: Controlli di sicurezza pre-commit.
- `auth-implementation-patterns`: Pattern sicuri per l'autenticazione.
- `secrets-management`: Gestione sicura dei segreti.
- `threat-modeling-expert`: Modellazione delle minacce.
